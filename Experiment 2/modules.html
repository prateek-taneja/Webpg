Modules

  • Home Page: to start with the website by selecting whether the user is a student or a company
  • Student Login: to login with username and password
  • Company Login: to login with company_id and password
  • Registration page: for storing the data of first time users
  • Resume builder: Students can fill up their details and upload certificates to create a resume
  • Job Description: A page where companies can fill their job descriptions and eligibility criterias
  • Dashboard: A dashboard where students can go for finding available jobs

System Requirements

  • Hardware Requirements
    • Laptop or Desktop - 1.6 GHz CPU
    • Memory: minimum 500mb RAM
    • Hard Disk Space - minimum 1gb


  • Software Requirements
    • Operating System - Windows or Linux
    • MySQL - 5.6 or more
    • Apache or XAMPP server
    • Net Browser - Chrome or FireFox
    • Code Editor - Notepad++
